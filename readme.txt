---------------Ceylon Tea---------------

Inorder to open the website open it via Live Server extension on Visual Studio Code or run the web application on a server. The website could also be accessed online via https://dexow.github.io

The index.html contains the home page therefore open it inorder to view the home page.

Inorder to view the Tour Booking system navigate to the Ceylon Tea Museum Page via the navigation bar (or enter this URL https://dexow.github.io/ceylon-tea-museum.html if viewing the hosted website) and scroll down to the Tickets Purchasing section.

To view the donation application navigate to the Shop & Donate page through the navigation bar (or enter this URL https://dexow.github.io/shop.html if viewing the hosted website)  and click the button "Donate" below the page description in order to view the donation options.
